🌸 Iris Flower Clustering using K-Means & PCA
📌 Project Overview
This project demonstrates the use of K-Means Clustering and Principal Component Analysis (PCA) on the famous Iris Dataset. The objective is to group iris flowers into clusters without using their actual labels and reduce the dataset dimensions for better visualization.
🚀 Objectives
Apply K-Means Clustering (k=3) on the Iris dataset.
Visualize the generated clusters.
Apply PCA to reduce the dataset from 4 dimensions to 2 dimensions.
Compare predicted clusters with the true class labels.
📂 Dataset
Dataset: Iris Dataset
Samples: 150
Features:
Sepal Length
Sepal Width
Petal Length
Petal Width
Classes:
Setosa
Versicolor
Virginica
🛠️ Technologies Used
Python

Scikit-learn

Install the required libraries using:
pip install numpy pandas matplotlib scikit-learn
🔍 Project Workflow
Load the Iris dataset.
Apply K-Means clustering (k=3).
Print cluster centers.
Visualize clusters using Matplotlib.
Apply PCA to reduce dimensions to 2D.
Calculate the explained variance ratio.
Compare predicted clusters with the actual labels.
📊 Results
Successfully clustered the Iris flowers into 3 groups.
PCA reduced the dataset from 4 dimensions to 2 dimensions while preserving most of the information.
Setosa was clustered almost perfectly, while Versicolor and Virginica showed slight overlap.
