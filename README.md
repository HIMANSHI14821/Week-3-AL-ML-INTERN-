# 🌸 Week 3 - AI/ML Internship: Clustering & Dimensionality Reduction

## 📖 Overview
This repository contains the **Week 3** tasks completed as part of my **AI/ML Internship**. It covers the implementation of **K-Means Clustering** and **Principal Component Analysis (PCA)** using the Iris Dataset.

The project demonstrates how unsupervised machine learning algorithms can group similar data points and how PCA helps reduce dataset dimensions for visualization and analysis.

---

## 📂 Repository Contents

- 📄 Iris.csv
- 📘 Practice+Assignment questions.ipynb
- 📘 MINI PROJECT-3.ipynb

---

## 🎯 Objectives

- Apply K-Means Clustering on the Iris dataset.
- Print cluster centers.
- Visualize clusters using Matplotlib.
- Apply PCA to reduce the dataset from 4D to 2D.
- Calculate the explained variance ratio.
- Compare predicted clusters with actual labels.

---

## 📊 Dataset

**Dataset:** Iris Dataset

- 150 Samples
- 4 Features
- 3 Classes
  - Setosa
  - Versicolor
  - Virginica

---

## 📌 Project Workflow

1. Load the Iris Dataset.
2. Perform K-Means Clustering (k=3).
3. Display Cluster Centers.
4. Visualize the clusters.
5. Apply PCA for dimensionality reduction.
6. Plot the PCA-transformed data.
7. Compare predicted clusters with actual labels.

---

## 📈 Results

- Successfully clustered the Iris dataset into three groups.
- PCA reduced the dataset to two principal components while preserving most of the information.
- Cluster visualization clearly shows the separation among flower species.
